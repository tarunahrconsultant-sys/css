1 Definition: Web development is the process of creating, building, and maintaining websites and applications for the internet or an intranet.
2 Front-End: This involves client-side development, focusing on the user interface (UI), layout, and user experience (UX) within the browser.
3 Back-End: This server-side development manages the behind-the-scenes logic, databases, security, and application functionality.
4 Core Technologies: HTML (structure), CSS (styling), and JavaScript (interactivity) are the foundational languages used to build web pages.
5 Databases: Websites often use relational (SQL) or non-relational (NoSQL/MongoDB) databases to store and retrieve user data.

6 Full-Stack: Full-stack developers possess skills to manage both front-end and back-end aspects of a project.
7 Responsive Design: A crucial practice that ensures websites look and perform optimally on various devices and screen sizes.
8 Frameworks/Libraries: Modern development utilizes tools like React, Angular, Vue, or backend frameworks like Node.js and Django to speed up development.
9 Version Control & Deployment: Git is used for managing code changes, while deployment publishes the final site for access, often through automated workflows.
10 Maintenance & Optimization: Continuous work is required to ensure performance, security, and SEO, keeping the website functional and updated.
